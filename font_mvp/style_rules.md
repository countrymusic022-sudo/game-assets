# 「銀河 Leo 星王」MVP 字體風格規則

> 目標：建立可量產的字型規則，而非單張海報字樣的像素拷貝。

## 1) 視覺定位

- 關鍵詞：**神話、星際、王者、金屬雕刻**。
- 風格類型：Display Title Font（標題展示字）。

## 2) 幾何與比例

- UPM：1000
- 建議 x-height（拉丁）：480–520
- 拉丁大寫高度：680–740
- 中文字面：860–900（接近滿框）
- 預設字重：Bold~Black

## 3) 筆畫語言

- 主筆畫直線為主，降低過度書寫感。
- 轉角偏銳利，避免圓潤卡通感。
- 收筆可加入「切角」或「尖刺」語彙。
- 局部可保留刀鑿缺口，模擬雕刻痕跡。

## 4) 明暗與材質（在排版層處理）

- 字型本體以純輪廓為主。
- 金屬質感由後製完成：
  - 漸層（冷銀→暖金高光）
  - 內陰影（增加厚度）
  - 外光暈（星塵氛圍）

## 5) MVP 字集策略

- 第 1 階段：品牌詞核心字（如「銀河Leo星王」）
- 第 2 階段：A–Z / a–z / 0–9 / 常用標點
- 第 3 階段：中文常用 300 字

## 6) 品質檢查

- 小尺寸 24px 可辨識
- 標題尺寸 72px 以上有風格張力
- 字距（tracking）在 -10 ~ +20 間都可用
